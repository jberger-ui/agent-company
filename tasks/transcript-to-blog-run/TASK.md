# Transcript to Blog Run

## Trigger condition

This task should only be used when the task payload includes a transcript, transcript summary, call notes with transcript evidence, or another explicit transcript-derived source bundle.

## Standard stages

1. summarize
2. route ideas
3. select viable opportunities
4. shape for persona
5. check overlap
6. draft
7. review for approval
8. publish only after approval

## Required outputs

- run summary
- accepted ideas list
- rejected ideas list
- persona briefs
- overlap reports
- draft artifacts
- approval status
- publish status if applicable
