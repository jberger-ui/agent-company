# QA Release Gate

## Checklist

- approval exists
- overlap recommendation is acceptable
- title and slug are valid
- metadata is complete
- draft aligns with intended search and editorial purpose
- no preventable factual overclaiming remains

## Deliverable

A release decision per draft:

- approved for publication
- approved with required metadata fixes
- blocked pending revision
- rejected
